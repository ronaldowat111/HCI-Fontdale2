exists, the page shows a tasteful gradient placeholder (the broken `<img>` is
auto-removed via `onerror`), so the layout looks intentional either way.

## How to export from Figma
Select the image layer (or its frame) → right panel **Export** → `1x` `JPG`
(or PNG) → **Export**. Rename to match the list above and place in this folder.
